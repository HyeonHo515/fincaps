# CodeMind

PDF 학습 자료를 요약하고, 요약 내용을 바탕으로 빈칸 문제와 AI 역질문을 만들어 복습할 수 있는 학습 보조 웹 애플리케이션입니다. 커뮤니티, 채널 채팅, 코딩 연습 기능도 함께 포함되어 있습니다.

## 주요 기능

- PDF 또는 텍스트 기반 학습 자료 요약
- PDF 안의 이미지, 표, 그래프 등 시각 자료 추출
- 텍스트 요약과 시각 자료 요약을 구분해서 구조화 요약에 표시
- 요약 내용을 기반으로 빈칸 문제 생성
- AI 역질문과 설명 평가 기능
- 요약 노트, 오답노트, 학습 메모 저장
- 커뮤니티 게시판과 채널 채팅
- 코딩 연습 및 실행 결과 확인

## PDF 요약 흐름

1. 파일 불러오기
   - PDF 텍스트를 추출합니다.
   - PDF 내부에 포함된 이미지 자료를 추출해 시각 자료 영역에 표시합니다.

2. 요약 생성
   - 추출된 텍스트를 구조화 요약으로 정리합니다.
   - 이미지, 표, 그래프에서 분석된 내용은 시각 자료 요약으로 따로 구분해 표시합니다.

3. 요약 저장
   - 텍스트 요약, 시각 자료 목록, 시각 자료 요약, 생성된 문제 정보를 함께 저장합니다.

## 실행 방법

```bash
./gradlew.bat bootRun
```

브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:8080
```

## 설정

OpenAI API를 사용하는 요약 기능을 실행하려면 `src/main/resources/application.properties`에 API 키와 DB 설정이 필요합니다.

`application.properties`에는 개인 API 키나 DB 비밀번호가 들어갈 수 있으므로 GitHub에 올리지 않습니다.

예시:

```properties
openai.api.key=your-api-key
```

## 주의 사항

- `uploads/` 폴더는 사용자가 업로드한 파일이 저장되는 위치이므로 GitHub에 포함하지 않습니다.
- PDF 안의 시각 자료가 실제 이미지 파일로 포함된 경우 추출됩니다.
- 표나 그래프가 텍스트, 선, 도형으로만 구성된 PDF라면 이미지 객체로 추출되지 않을 수 있습니다.
